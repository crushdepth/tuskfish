TUSKFISH RESPONSIVE DESIGN
==========================
* Will design for two size classes of device:
  - Viewports <= 992 px (phones and tablets), corresponding to medium viewport and below.
  - Viewports >= 992 px (desktops and wide view screens), corresponding to large viewport and above.

* Not going to address all size classes, mobile devices are quite flexible now with zoom behaviour.
  - Nexus 5: 1080 x 1920
  - Nexus 7: 1920 x 1200
  - iPhone 7: 1334 x 750 or 1920 x 1080 
  - Galaxy Note 7: 1440 x 2560
