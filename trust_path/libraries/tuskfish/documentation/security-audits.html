Standard code security audit

All PHP files *must* be audited against the following criteria:

  • Externally supplied data ($_POST, $_GET etc) may only be accepted for processing if it's on a whitelist of expected data and range checked as far as possible prior before use.
  • Database operations exclusively use prepared statements (PDO) with bound parameters to guard against SQL injection. No parameter may be directly included in an SQL statement; all parameters are required to be inserted via a bound parameter or bound value.
